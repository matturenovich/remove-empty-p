== Description ==

Remove empty paragraph tags, and broken paragraph tags caused by shortcodes with content.

[github](https://github.com/ninnypants/remove-empty-p)

== Installation ==

1. Upload `remove-empty-p.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
